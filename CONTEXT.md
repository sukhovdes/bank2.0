# Контекст проекта — Прототип Ozon Банк для бизнеса

> Хендофф-файл для продолжения работы в новом окне контекста. Обновлять по ходу.

## Что это
Кликабельный прототип кабинета Ozon Банка для бизнеса. **Главная гипотеза — нижний тулбар-док в стиле macOS** (проверяем юзабилити). Делается для презентации/демо.

## Стек и запуск
- React 18 + Vite 5, обычный CSS (НЕ Tailwind). Доп: `motion` (анимации), `vaul` (drawer).
- Запуск: `npm run dev` → http://localhost:5173 (держим фоновым процессом: `nohup npm run dev > /tmp/vite-prototype.log 2>&1 &`).
- Сборка: `npm run build` → `dist/`.

## Структура кода
- `src/App.jsx` — оболочка: шапка (логотип-кнопка на главную, AiSearch по центру, колокольчик+шестерёнка+профиль-пилюля), верхний баннер «Ограничения на счете», `<main>`, `<Dock>`. Состояния: view, products, banner. Welcome-экран первым (`entered`).
- `src/Welcome.jsx` — приветствие: пульсирующее синее сердце + blur-reveal текст «Привет, это новый Ozon Банк для бизнеса», автопереход через 200ms.
- `src/components/BlurReveal.jsx` — адаптация spell.sh blur-reveal на `motion`.
- `src/Dock.jsx` — тёмное стекло (#003078, 38%, blur), разделы (Главная/Платежи/Сервисы, светлый текст, активный — белая пилюля), продукты-картинки (drag вверх = удалить), «+» → Сервисы. CSS-тултип у иконок.
- `src/pages/Home.jsx` — Главная. Быстрые действия (настраиваются шестерёнкой → `QuickSettings`), грид: левая колонка продуктов + правая (заявка на кредит с анимацией прогресса, Задачи, История).
- `src/components/HomeCards.jsx` — `BusinessAccountCard` (клик по телу → баннер, по иконке карты → CardModal), `DeferralCard` (Отсрочка), `PromoCard` (Счёт ежедневных выплат, закрывается).
- `src/components/CardModal.jsx` — модалка карты (tilt-card spell.sh) с реальными ассетами (логотип, сердце, облако, МИР `mir.svg`).
- `src/components/QuickSettings.jsx` — модалка настройки быстрых действий: тумблеры + drag-перетаскивание (drag-подсветка через `outline`).
- `src/components/AiSearch.jsx` — поиск в шапке: меняющийся плейсхолдер + модалка с подсказками AI (звёздочки `sparkles.svg`).
- `src/components/PaymentDrawer.jsx` — vaul Drawer «Платёж отправлен» (галочка-анимация), на всю ширину, высота по контенту, кнопки 260×56.
- `src/pages/Payments.jsx` — лево-меню + форма перевода («Платежи и переводы»); клик по «Получатель» заполняет мок-данными; «Подписать и отправить» → PaymentDrawer.
- `src/pages/Services.jsx` — 10 продуктов (картинки `public/products/*.png`, ascii-имена), вёрстка: картинка слева + заголовок + описание, 2 колонки.
- `src/pages/Product.jsx` — страница неподключённого продукта: без меню, контейнер по центру.
- `src/styles.css` — все стили. `src/icons.jsx` — часть иконок (см. долг ниже).

## Превью-нюансы (важно при проверке)
- headless-превью даёт `window.innerWidth=1` иногда → координаты/hover недостоверны; ресайзить до 1366×900.
- `backdrop-filter` в превью-скриншот НЕ рендерится (в реальном браузере ок).
- fixed-док иногда «пропадает» в скриншоте при скролле — артефакт превью.
- Относительные пути в Write/Edit резолвятся от cwd Bash — после `cd public/...` обязательно вернуть cwd в корень, иначе файл уйдёт не туда (был баг с Services.jsx).
